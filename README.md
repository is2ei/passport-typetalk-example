passport-typetalk-example
===

## Usage

### Install

```shell
$ git clone git@github.com:is2ei/passport-typetalk-example.git
$ cd passport-typetalk-example
$ npm install
```

### Set config

```json
{
    "clientID": "TYPETALK_CLIENT_ID",
    "clientSecret": "TYPETALK_CLIENT_SECRET"
}
```

### Run

```shell
$ node .
```

Open http://localhost:3000
